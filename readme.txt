this data analysis project covers the exploration and analysis of Countries Financial Indicators.
Data: multiple indicators from the World Bank open database --.
Web-app: https://country-finance.herokuapp.com/
The app was created using Dash and deployed to heroku.
This repository contains 2 folders: 
  Data_Exploration: which containes a jupyter notebook covering the data cleaning and exploration stage, 
                    you can also find it on kaggle: https://www.kaggle.com/code/nabilmostafa/financial-data-analysis/notebookand
  Dash_Web_App: which is for the creation of the web application.
